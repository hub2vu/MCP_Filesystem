1. 의존성 설치
docker compose down
docker compose up --build
2. MCP 서버 키기
docker compose up mcp-server
3. 클라이언트 실행(새 터미널)
docker compose run --rm agent
4. web 실행
docker compose up mcp-server web
5. Global Paper Graph(내가 가지고 있는 논문) 
http://localhost:3000/
6. Paper Reference Graph(특정 논문의 레퍼런스 : 아래 코드의 마지막 10.48550_arxiv.2506.07976가 특정 논문)
http://localhost:3000/paper/10.48550_arxiv.2506.07976
